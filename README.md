# Tiny Tines

# Overview
A tines application 

# Features
- Development based on Nodejs.

# Stack 
  >  * Javascript
  >  * Nodejs 12.16
  >  * Jest 
  >  * Npm 6.14.4
  >  * Git

 
 # Run the project
  1. Clone

    ```bash
     $ git clone https://github.com/JayGibran/tiny-tines.git
     $ cd tiny-tines
     ```

  2. Run
  ```bash
    $ npm install
    $ npm start [jsonFilePath]
  ```
  3. Test
  ```bash
    $ npm test
  ```
